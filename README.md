# 🌅 Personal Portfolio UI (Early Stage)

Bu proje, web geliştirme serüvenimin başlarında tamamen **sıfırdan (hiçbir yapay zeka veya hazır şablon kullanmadan)** kendi el emeğimle tasarladığım ilk kişisel portfolyo arayüzüdür. Benim için hem nostaljik bir değeri var hem de frontend (HTML/CSS) temellerimi atarken harcadığım emeğin en somut kanıtıdır.

🔗 **[Sitenin Canlı Halini İncelemek İçin Tıklayın](https://emirhansartik.github.io/Portfolio-Site/)**

## 🚀 Proje Hakkında

*   **%100 El Yazması Kod:** Hiçbir CSS framework'ü (Bootstrap, Tailwind vb.) kullanılmadan, tamamen saf (vanilla) HTML ve CSS ile satır satır inşa edilmiştir.
*   **Kişisel Tasarım:** Renk paleti, yerleşim ve "Skills" (Yetkinlikler) barları tamamen kendi hayal gücüme göre tasarlanmıştır.
*   **Gelişim Göstergesi:** Sitede de yazdığım *"JavaScript bitince asıl eğlence başlıyor!"* mottosuyla yola çıktığım o ilk öğrenme heyecanımı yansıtır. Güncel yetkinliklerim ve yeni projelerim için GitHub ana profilimi ziyaret edebilirsiniz.

## 🛠️ Kullanılan Teknolojiler

*   **HTML5**
*   **CSS3**
